# webgl-visualiser

A simple WebGL audio visualiser.

### Prerequisites

- [Node.js](https://nodejs.org/)
- [Yarn](https://yarnpkg.com/)

### Installation

```bash
yarn install
```

### Development

Start the development server:

```bash
yarn dev
```

### Build

Build the project for production:

```bash
yarn build
```

### Preview

Preview the production build:

```bash
yarn preview
```
